Eat-Da-Burger

Eat Da Burger is a Bob's Burger themed app that utilizes Express, Handlebars, and SQL to allow users to create and eat the iconic burgers featured in the show.


Crystal Collector

Crystal Collector is a jQuery game with a randomly generated target number. The value of the crystals is randomized with each round and is only known to the users once they are clicked on. If the user's guess equals the random target number then the user wins.


Zoom

A app that allows users to store the vehicles that come into the shop for inventory and organization. The user can log each vehicle into the system and sort the vehicles by status. "WIP" (work in progress) which are cars that need to be repaired. "DONE/ READY TO SELL" (completed vehicles) cars that are done with repairs and have gone through a road test & ready to be sold to a customer.


Muttropolitan

The Muttropolitan Site is an all inclusive dog owner reference site. We offer guidelines on proper practices on caring for you furry four legged friends. Exploring the site, you will find links to near by resources such as Veterinarians and dog parks, as well as some shopping for poochie pampering. Feel like you need a new companion? Browse our list of dogs available for adoption.


Bamazon

The Bamazon application is an Amazon-like storefront that is run soley through. Users are given the choice of buying products as a customer or updating inventory a manager.


Giftastic

Giftastic is a web application that allows users to view gifs of their favorite artists. The application features preset options as well as the option to search for the users artist of choice.


Train Scheduler

The train scheduler application allows users to track trains by name, destination, frequency, next arrival, and by how many minutes away it is. Users are also able to input their own trains into the system if they are not already on the list. The schedule is built to utilize the current time for the most accurate schedules.
